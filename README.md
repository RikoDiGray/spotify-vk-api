# spotify-vk-api

Установка:
1. Установить NodeJs библиотеку Spotify и vk-io
2. Получить токен ВК и Spotify
3. Вставить в getMe.js токен ВК и Spotify (Большими буквами всё указано)
4. В index.js вставить clientId и clientSecret

# Ссылки:
